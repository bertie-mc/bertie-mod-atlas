# bertie · mod atlas

Interactive progression atlas for the **bertie** modpack (NeoForge 1.21.1).

Every mod is sorted into **content** (on the progression spine) vs **fluff** (outside it), and content is filed on a **3-stage spine** — S1 Adventure · S2 Machines & Power · S3 Void Factory — with addon ecosystems folded into families.

**Live site:** https://bertie-mc.github.io/bertie-mod-atlas/

- [`index.html`](index.html) — the interactive, filterable atlas (self-contained, no build step)
- [`bertie-mod-stages.md`](bertie-mod-stages.md) — the markdown source-of-record

Working draft — placements iterate as pack recipes get intertwined.
